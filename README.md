#AudioCloud

##Description
- Users can search for audio files. Users can also upload and play music on the website.

##Functionality & MVP
- User authentication
- Audio files (Create and Read only)
- Progress bar for audio playback
- Search for audio file
- Design Docs
- Wireframes
